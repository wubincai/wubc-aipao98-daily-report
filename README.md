# 爱跑98竞赛日报表生成器 (wubc-aipao98-daily-report)

自动生成加油站爱跑98竞赛日班组对抗赛和个人销量奖励统计表。

## 功能

- 读取加油交易记录明细（.xls 格式），按时间段汇总
- 生成双表 Excel：班组对抗赛统计表 + 个人销量奖励统计表
- 支持灵活配置时间段、班组数、班组人数
- 从数据源自动提取站名
- 基础奖励值可配置
- 配置持久化保存，无需重复填写

## 前置依赖

- Python 3.8+
- 依赖包: `xlrd`, `openpyxl`

安装依赖:
```bash
pip install xlrd openpyxl
```

## 使用方法

在 Codex 中调用此 skill，提供日期即可：

```
帮我生成7月10日的爱跑98竞赛日报
```

### 首次使用流程

1. 提供日期
2. 按引导配置时间段、班组数、人数、基础奖励
3. 确认数据源位置（默认 `~/Documents/加油站数据表/`）
4. 从管控系统下载交易记录明细并放入数据源目录
5. 自动生成报表保存至 `~/Documents/爱跑98竞赛日报表/`

### 后续使用

- 自动读取已有配置，也可要求修改
- 只需提供新日期即可生成

## 输出说明

生成的双表 Excel 包含：

**左表 — 班组对抗赛统计表**
- 各时间段汽油总升数、爱跑98升数及占比
- 按班组分布的成员人均数据
- 系数、基础奖励、个人奖励金额、班组奖励金额（公式自动计算）

**右表 — 个人销量奖励统计表**
- 紧凑去重组员名单
- 每人爱跑98升数汇总
- 系数匹配
- 个人奖励汇总

> 站长需在左表 F 列（组员）和 H 列（系数）填写数据后，所有公式自动运算。
